# Process & Instrumentation Diagrams (P&ID) Portfolio

Chemical & Process Engineer (M.Sc.)

This repository contains examples of professional Process & Instrumentation Diagrams (P&ID) developed using AutoCAD.

### Included examples:
- Batch, CSTR and Plug Flow Reactor system
- Distillation column with pressure, level and temperature control
- Vessels and separators with safety systems

The diagrams demonstrate industrial-standard control strategies, cascade control, and integrated safety philosophy.
